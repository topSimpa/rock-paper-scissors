# Rock Paper Scissors
This program mimicks the popular Rock, paper, Scissors.
It is a fun and easy hand game that anyone can learn and enjoy. This game is created as part of the TheOdinProject foundation Requirement

## Objectives
- To create basic Javascript function
- To demonstrate skills in comparisons in Javacript
- To demonstrate skills in mathematical computation
- To show mastery of Comparisons in Javascript
- To build a javascript program that can run on the web


**As of 2nd April 2026, this project is not meant to have a GUI**
